# Nural_EA - AI-Driven Scalping Expert Advisor

This project is an implementation of the "Enhanced AI-Driven Scalping EA Architecture". It's a sophisticated trading bot that uses a multi-layered architecture for data ingestion, AI-based signal processing, and trade execution on MetaTrader 4.

## Architecture Overview

The architecture is composed of five main layers:

1.  **Data Ingestion:** Collects market data from various sources.
2.  **AI Processing Pipeline:** Analyzes data using a multi-agent AI system to generate trading signals.
3.  **Ultra-Low Latency Bridge:** Transmits signals to the execution layer with low latency.
4.  **Execution Layer (MT4):** An MQL4 Expert Advisor that manages trades.
5.  **Monitoring & Visualization:** Dashboards for real-time monitoring of the system.
